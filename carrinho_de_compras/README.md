# Carrinho de Compras em Python

Projeto desenvolvido por mim em Python utilizando Programação Orientada a Objetos (POO),
simulando um carrinho de compras de lojas online.

## Funcionalidades
- Adicionar produtos ao carrinho
- Remover produtos
- Listar produtos
- Calcular o total da compra
- Limpar carrinho
- Salvar e carregar o carrinho em arquivo JSON
