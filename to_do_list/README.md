# Lista de Tarefas em Python

Projeto desenvolvido por mim em Python utilizando Programação Orientada a Objetos (POO), simulando um gerenciador de tarefas para organização pessoal.

## Funcionalidades
- Adicionar tarefas à lista
- Concluir tarefas
- Listar tarefas
- Editar descrição das tarefas
- Remover tarefas
- Salvar e carregar a lista em arquivo JSON
