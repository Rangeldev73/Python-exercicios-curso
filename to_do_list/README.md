#Lista de Tarefas em Python

Projeto desenvolvido por mim em Python utilizando Programação Orientada a Objetos (POO), simulando um gerenciador de tarefas (To-Do List) com persistência de dados.

##Funcionalidades
- Adicionar tarefas com descrição personalizada
- Marcar tarefas como concluídas (alterando o status para [x])
- Remover tarefas da lista através do índice
- Editar a descrição de tarefas já existentes
- Listar todas as tarefas com exibição de status e total de itens
- Salvar e carregar a lista automaticamente em um arquivo JSON
